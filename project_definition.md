Cel:
Zbudować model klasyfikacyjny przewidujący churn klientów telco,
ze szczególnym uwzględnieniem różnic między klientami o różnym tenure
(tenure buckets), aby umożliwić wcześniejsze wykrywanie churnu
i ograniczenie strat finansowych.

Założenia biznesowe:

False negatives (niewykryty churn) są bardziej kosztowne niż false positives

Klienci z długim tenure mają wyższy LTV

Ewaluacja musi być globalna i per bucket
